Harbour Wiki Static Article Template

Migration template. Replace this page with the first real migrated article after the article layout is approved.

This page demonstrates the simplified static article model. The article index opens each article in a new tab using JavaScript. Because the tab was script-opened, the Close button can normally close the article tab.

If the article was opened directly from a bookmark, pasted URL, or browser history, the browser may not allow JavaScript to close it. In that case, use Back to Articles.

Article file convention

articles/<article-slug>/index.html
articles/<article-slug>/images/

Image convention

Use normal image sources, not lazy-loaded images, so printing and print-to-PDF are reliable.

<img src="images/example.png" class="img-fluid border rounded" alt="Descriptive image text">
