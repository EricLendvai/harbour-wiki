The New Buccaneers — Episode 2 by Manu Expósito Suárez

In 2021, Harbour developer Manu Expósito Suárez released one of the most valuable educational works ever written for the Harbour community:

Los nuevos bucaneros. Episodio 2. Lenguaje C para programadores Harbour
(The New Buccaneers, Episode 2 – The C Language for Harbour Programmers)

Although written several years ago, this book remains as relevant today as the day it was published. Harbour’s architecture has stabilized and matured, and because its C interfaces and Virtual Machine internals have remained consistent, every topic in this book continues to apply perfectly to modern Harbour development.

Special thanks to Manu for allowing the translation and publication of your book!

A Practical Guide to Harbour’s Inner Workings

The book is not just about the C language — it’s about understanding Harbour from the inside out.

Through clear explanations and practical examples, Manu Expósito shows how Harbour’s Virtual Machine (HVM) works, how the stack, symbol table, and Item API interact, and how developers can use these concepts to extend Harbour’s functionality with native C code.

This combination makes the book both a technical reference and a hands-on guide. It demystifies Harbour’s core — what really happens when you call a function, push parameters onto the stack, or pass values between PRG and C — and then shows how to build real working examples that take advantage of this internal design.

Why This Knowledge Still Matters

Unlike many other programming ecosystems, Harbour has achieved long-term stability. Its APIs — including the Extended System, Item API, and Virtual Machine interfaces — have not required major breaking changes for years.

That stability means that a book written in 2021 is still a current and reliable reference for any developer who wants to explore:

  • How to call C functions directly from Harbour,
  • How to expose Harbour procedures to be invoked from C,
  • How to work with codeBlocks, arrays, and hash tables across both languages,
  • How to build static and dynamic libraries (.lib, .a, .dll, .so),
  • And how to safely interface Harbour code with external libraries and APIs.

In essence, this book turns Harbour into a language that can stand shoulder to shoulder with C, Python, or Go when it comes to performance and extensibility.

Extending Harbour with C — A Bridge to the Modern World

For many Harbour developers, performance or external integration has often been the point where pure PRG code reaches its limit.
That’s where this book becomes a bridge — a manual on how to go beyond the PRG layer safely and efficiently.

By applying the techniques explained by Manu Expósito Suárez, it becomes possible to:

  • Embed high-performance routines written in C directly into Harbour applications.
  • Integrate existing C or C++ libraries — for example, graphics engines, encryption toolkits, database connectors, or machine-learning models.
  • Create hybrid applications where Harbour acts as the business logic layer while C handles compute-intensive tasks.
  • Build modules that serve as communication layers between Harbour and other languages.

The possibilities are nearly endless. Once you understand how Harbour’s stack, symbols, and Items interact with C structures, you realize that any C-compatible language (Python, Rust, Go, or even Java via JNI) can become interoperable with Harbour.

Real-World Applications

Several modern Harbour projects have already benefited from these ideas.

For instance, Eric Lendvai (maintainer of harbour.wiki and several open-source Harbour projects) applied the same principles from this book to:

  • Integrate CPython into Harbour, allowing native Python functions and modules to be called directly from Harbour code. Many thanks to Antonio Linares for opening the doors for CPython integration into Harbour!
  • Develop a FastCGI-based Harbour web server, enabling high-performance web applications written entirely in Harbour.
  • Create seamless C-based bridges that allow Harbour to interact with other runtimes or technologies — effectively extending Harbour’s reach far beyond its original scope.

These examples demonstrate the long-term potential of Manu Expósito’s work. His book is not just historical documentation; it is a practical toolkit that continues to inspire real innovation across the Harbour ecosystem.

Inside the Book

The English translation of The New Buccaneers — Episode 2 preserves all 21 chapters of the original Spanish edition, including:

  1. The fundamentals of the C language for Harbour programmers
  2. Operators, control structures, and data types in C
  3. How Harbour represents variables internally (HB_ITEM, stack management, and symbol tables)
  4. Creating and calling C functions from PRG code
  5. Passing parameters and returning values between C and Harbour
  6. Understanding and extending the Item API
  7. Working with arrays, hashes, and objects across both languages
  8. Handling errors and files through Harbour’s C APIs
  9. Building and linking static and dynamic libraries
  10. Interfacing with DLL functions from Harbour applications

Each topic is reinforced with practical examples and complete working code — including both Harbour (.prg) and C (.c) source files. These examples can be compiled directly with hbmk2, Harbour’s multi-compiler build tool.

A Lasting Contribution to the Harbour Community

The New Buccaneers, Episode 2 is more than a book — it is a lasting contribution to the Harbour ecosystem.
It connects the present with the roots of Clipper and the future of multi-language integration.

For any developer who wants to understand Harbour deeply or extend its capabilities beyond what is possible in pure PRG, this book remains the definitive guide.
Even years after its publication, it continues to empower programmers to explore, extend, and modernize Harbour applications with confidence.

Manu Expósito Suárez showed us that Harbour is not an island — it’s a gateway.
Through C, it connects to the entire world of programming.
